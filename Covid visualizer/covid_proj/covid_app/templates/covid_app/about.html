About this website?

Covid - Python package to get information regarding the novel corona virus provided by Johns Hopkins university and worldometers.info

COVID19Py - A tiny Python package for easy access to up-to-date Coronavirus (COVID-19, SARS-CoV-2) cases data.

NewsAPI - Get breaking news headlines, and search for articles from news sources and blogs all over the web.
